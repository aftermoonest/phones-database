8.	Написати аплікацію для зберігання каталогу мобільних пристроїв (телефонів та смартфонів)
1)	Створити клас “Smartphone” для збереження інформації про смартфон:
    1	Рік виробництва
    2.	Виробник
    3.	Модель
    4.	Розмір пам’яті'
    5.	Діагональ екрану
    6.	Колір
2)	Створити клас “Phone” для збереження інформації про телефон:
    1.	Рік виробництва
    2.	Виробник
    3.	Модель
    4.	Колір
    5.	Кількість сім-карт
3)	Для  цих класів створити конструктор за замовчуванням, конструктор з параметрами, конструктор копій;
    перевизначити операції >>, << для зчитування та запису у файл.
4)	Реалізувати функцію для пошуку мобільних пристроїв по назві виробника та роком виробництва.
5)	Визначити та вивести мобільні пристрої, з однаковим кольором та роком виробництва
6)	Інформація про мобільні пристрої повинна зберігатися та зчитуватися у файли smartphones.dat та mobilephones.dat відповідно
